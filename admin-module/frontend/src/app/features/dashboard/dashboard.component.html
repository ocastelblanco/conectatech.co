Sobre este panel

Este panel permite gestionar la plataforma educativa ConectaTech desde una interfaz web. Todas las operaciones actuan sobre la misma base de datos de Moodle y requieren que hayas iniciado sesion como administrador. Las operaciones de creacion y matriculacion son idempotentes: puedes ejecutarlas varias veces sin generar duplicados.
